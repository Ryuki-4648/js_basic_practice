JavaScriptの型は7種類

  • undefined
  • null
  • 数値

  • シンボル

3種類の変数宣言

  • var
  • let
  • const

var

現在ではvarを使う理由はない。
関数スコープおよび巻き上げのせいで、挙動の予測が困難になるから。

varを使うのは、古いブラウザを対象にした場合のみ。

let

letで宣言した変数はブロックスコープになる。
巻き上げは行われるが、宣言前に参照するとエラーとなる。

const

constは再代入不可能な変数を宣言する。
letと同じくブロックスコープで、巻き上げは行われるが宣言前の参照がエラーになる点も同じ。
ほとんどの変数は再代入の必要が無いので、基本的にはconstを使うほうがいい。
